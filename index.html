HTML Basics

HTML stands for HyperText Markup Language, the standard for creating webpages.

It is made up of elements represented by tags.

Tags usually come in pairs, with opening and closing tags.

HTML describes the structure of web pages.

Here’s an example of HTML:

<h1>Hello World</h1>

Some common HTML tags:

  • <p> - Paragraph
  • <h1> to <h6> - Headings
  • <a> - Links
<p>This is a paragraph</p>

CSS Basics

CSS stands for Cascading Style Sheets.

It is used to style HTML elements.

You can use inline, internal, or external CSS.

CSS uses selectors and properties.

Example CSS rule:

p { color: blue; }

Common CSS selectors:

  • Element selector
  • Class selector
  • ID selector
.title { font-size: 24px; }

JavaScript Basics

JavaScript is a programming language for the web.

It can be used to make pages interactive.

You can write JS inside <script> tags or in an external file.

JavaScript uses variables, functions, and events.

Example function:

function greet() { alert('Hello!'); }

Common JavaScript data types:

  • String
  • Number
  • Boolean
let x = 5;

Responsive Design

Responsive design ensures websites look good on all devices.

It uses flexible layouts and media queries.

CSS media queries apply styles based on device width.

Example media query:

@media (max-width: 600px) { body { font-size: 14px; } }

Key techniques:

  • Fluid grids
  • Flexible images
  • Viewport meta tag
img { max-width: 100%; height: auto; }

Deployment

Deployment is making your website live on the internet.

Common hosting platforms include GitHub Pages, Netlify, and Vercel.

Steps for deployment:

  • Prepare your files
  • Choose a hosting platform
  • Upload or push your files

Example GitHub Pages setup:

git push origin main

Once deployed, share your site’s link with others.

https://username.github.io/project-name
